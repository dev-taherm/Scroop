## Description

An online shopping application built using Next.js and Firebase.



## Features

- Authentication
- Wishlist items
- Add items to cart
- Sort items
- Filter items
- Real-time updates

## Tools

- [Next.js](https://nextjs.org)
- [React](https://reactjs.org)
- [Styled components](https://www.styled-components.com)
- [Firebase](https://firebase.google.com)
